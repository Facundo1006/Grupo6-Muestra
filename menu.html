Virtualización

Una máquina virtual (MV) es un contenedor de software aislado que simula un entorno de hardware completo, permitiendo la instalación de un sistema operativo (SO) dentro de ella. Cada máquina virtual opera de manera independiente, lo que significa que múltiples MVs pueden coexistir en una misma computadora física, conocida como servidor físico o host. Este aislamiento garantiza que las operaciones y configuraciones de una MV no afecten a las demás, lo que proporciona un entorno seguro y controlado para diversas tareas

La virtualización es una tecnología que permite ejecutar múltiples sistemas operativos y aplicaciones en un solo servidor físico, optimizando el uso de recursos y mejorando la eficiencia. Proporciona entornos aislados para cada aplicación, garantizando que los problemas en uno no afecten a los demás, lo que aumenta la seguridad y estabilidad. Además, facilita el desarrollo y pruebas de software al ofrecer entornos seguros y rápidos de configurar. La virtualización también simplifica la recuperación ante desastres y reduce costos al disminuir la necesidad de hardware adicional.

Tipos de vitualizacion

De servidores

Utiliza un hipervisor para participar los servidores fisicos en multiples servidores virtuales, cada uno con su propio sistema operativo.

De redes

Se divide el ancho de banda disponible en canales independientes, cada uno de los cuales se puede asignar a un servidor o a un dispositvo segun se nesecite.

De aplicaciones

Pueden ejecutar aplicaciones de una forma independiente del sistema operativo que se esté utilizando.

De almacenamiento

El almacenamiento físico de múltiples dispositivos en una red se une en un conjunto en un dispositivo de almacenamiento virtual unificado que se administra desde una consola central.
